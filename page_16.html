Benchmark Datasets

Standardized datasets for evaluating RAG systems across different task types and complexity levels.

Natural Questions (NQ)

  • Large-scale real-world QA dataset
  • Questions from Google search logs, answers from Wikipedia
  • Primary for single-hop QA task evaluation

MS MARCO

  • Microsoft Machine Reading Comprehension
  • Real user queries and documents from Bing search results
  • Used for information retrieval and document ranking

HotpotQA

  • Multi-hop QA dataset requiring complex reasoning
  • Information integration from multiple documents
  • Evaluates RAG systems' reasoning capabilities

ELI5 (Explain Like I'm Five)

  • Long-form QA dataset with detailed explanations
  • Questions require background knowledge
  • Evaluates long-form, explanatory answer generation
