Evaluation Metrics

Assessing RAG systems requires metrics across multiple dimensions to evaluate both retrieval and generation quality.

Retrieval Metrics

Recall

Measures the retriever's ability to find all relevant documents. High recall ensures LLM has sufficient information.

Precision

Measures the proportion of relevant documents among retrieved results. High precision reduces noise for LLM.

Generation Metrics

Faithfulness

Evaluates if generated content is fully based on retrieved evidence. Core measure of RAG's hallucination reduction.

Answer Relevance

Measures direct correlation between generated answer and user query. Critical for task performance.

Information Integration

Assesses ability to synthesize information from multiple documents for complex queries.

System Metrics

Latency

Time from query receipt to final response. Critical for real-time applications like chatbots.

Noise Robustness

System's ability to maintain performance with irrelevant or misleading documents in knowledge base.
